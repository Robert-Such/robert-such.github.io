Intro

In this project we will take some inspiration from one of Alex Freberg's excellent portfolio project suggestions (see here). There will be some deviations in my approach due to differences in my setup such as being able to pull data into Tableau from SQL server directly. Here are some of the specifics of my setup:

  • My main machine is an intel based mac (macOS 12.2)
  • The database I am using is MS-SQL Server 2019 (Express), made available through a docker container.
  • Visualizations are generated through Tableau desktop, installed locally (ver. 2021.4.3 64-bit)
  • DataGrip is what I am using for the database management environment, installed locally (ver. 2021.3.4) although Azure Data Studio would work fine as well.

Obtaining the Data

Our Covid-19 dataset is being sourced from OWID (Our World in Data) which maintains this data on a daily basis. The data can be downloaded from the github page (here). In the readme section of this page you will see an option to download as a CSV, XLSX, or JSON. In this case we will be downloading as a CSV.
